系统要求

确保您的环境满足以下要求

Java环境

  • Java 8或更高版本
  • JRE/JDK 1.8+
  • 支持OpenJDK和Oracle JDK

Excel支持

  • Apache POI 5.2.3+
  • 支持.xls和.xlsx格式
  • 支持Excel 97-2019

后端数据库

  • HSQLDB 2.7.2+
  • MySQL 8.0.33+
  • PostgreSQL 14+ (开发中)

安装指南

快速开始使用xlSQL

手动安装

  1. 下载xlSQL驱动JAR文件
  2. 将JAR文件添加到项目classpath
  3. 添加必要的依赖项(POI等)
  4. 按照文档配置连接参数

Maven安装

在pom.xml中添加以下依赖:

<dependency>
    <groupId>com.jsdiff</groupId>
    <artifactId>xlsql</artifactId>
    <version>3.0-SNAPSHOT</version>
</dependency>
