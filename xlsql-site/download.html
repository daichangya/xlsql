下载安装

获取最新版本的 XLSQL，开始您的数据集成之旅。

系统要求

  • Java SDK: JDK 8 或更高版本
  • Maven: 3.6.0 或更高版本
  • 操作系统: Windows, Linux, macOS

Maven Central（推荐）

XLSQL 5.1.1 已发布到 Maven Central！ 您可以直接通过 Maven 依赖使用，无需手动下载。

在您的 pom.xml 中添加以下依赖：

<dependency>
    <groupId>io.github.daichangya</groupId>
    <artifactId>xlsql</artifactId>
    <version>5.1.1</version>
</dependency>

在 Maven Central 查看

从源码构建

如果您需要从源码构建或获取最新开发版本：

终端命令
1. 克隆仓库
git clone https://github.com/daichangya/xlsql.git
cd xlsql
2. 编译和测试
mvn clean package
3. 安装到本地仓库
mvn install

构建产物说明

构建完成后，target/ 目录下会生成以下文件：

标准 JAR

xlsql-5.1.1.jar

仅包含 XLSQL 核心代码。使用时需要自行在 classpath 中添加所有依赖库。

适合 Maven 项目
Shaded JAR (Fat JAR)

xlsql-5.1.1-shaded.jar

包含所有依赖库（POI, HSQLDB等）。可以直接作为单独的库使用，无需额外配置依赖。

适合非 Maven 项目

手动安装 JAR

如果您需要将 Shaded JAR 手动安装到本地 Maven 仓库：

mvn install:install-file \
  -Dfile=target/xlsql-5.1.1-shaded.jar \
  -DgroupId=io.github.daichangya \
  -DartifactId=xlsql \
  -Dversion=5.1.1 \
  -Dpackaging=jar
