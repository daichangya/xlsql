快速开始

系统要求
  • Java 8或更高版本
  • Apache POI 5.2.3或更高版本
  • HSQLDB 2.7.2或MySQL 8.0.33(作为后端数据库)

安装步骤

  1. 下载xlSQL驱动JAR文件
  2. 将JAR文件添加到项目classpath
  3. 添加必要的依赖项(如POI、HSQLDB等)
  4. 使用标准JDBC API连接Excel文件

Maven配置

<dependency>
    <groupId>com.jsdiff</groupId>
    <artifactId>xlsql</artifactId>
    <version>4.0-SNAPSHOT</version>
</dependency>

代码示例

基本查询

// 注册驱动
Class.forName("com.jsdiff.xlsql.jdbc.xlDriver");

// 创建连接
String url = "jdbc:jsdiff:excel:/path/to/excel/files";
Connection conn = DriverManager.getConnection(url);

// 执行查询
Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM \"test2.Sheet1\"");

// 处理结果
while (rs.next()) {
    System.out.println(rs.getString(1));
}

// 关闭资源
rs.close();
stmt.close();
conn.close();

参数化查询

// 使用PreparedStatement
String sql = "SELECT * FROM \"test2.Sheet1\" WHERE department = ?";
PreparedStatement pstmt = conn.prepareStatement(sql);
pstmt.setString(1, "Sales");

ResultSet rs = pstmt.executeQuery();
while (rs.next()) {
    // 处理结果
}

批量插入

// 批量插入数据
conn.setAutoCommit(false);
String sql = "INSERT INTO \"test2.Sheet1\" (name, age) VALUES (?, ?)";
PreparedStatement pstmt = conn.prepareStatement(sql);

pstmt.setString(1, "张三");
pstmt.setInt(2, 30);
pstmt.addBatch();

pstmt.setString(1, "李四");
pstmt.setInt(2, 28);
pstmt.addBatch();

int[] counts = pstmt.executeBatch();
conn.commit();
